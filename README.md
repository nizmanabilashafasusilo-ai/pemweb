# Golden Wave

Golden Wave adalah aplikasi manajemen reservasi akomodasi berbasis web yang dikembangkan untuk memudahkan proses pemesanan kamar, pengelolaan reservasi, serta administrasi data oleh pengelola. Aplikasi ini dibuat sebagai bagian dari tugas proyek mata kuliah Pemrograman Web.

## Deskripsi Singkat

Aplikasi Golden Wave menyediakan layanan reservasi akomodasi secara online yang dapat diakses oleh pengguna umum dan administrator. Pengguna dapat melakukan pemesanan kamar, melihat riwayat reservasi, melakukan check-in online, serta mengajukan pembatalan reservasi. Sementara itu, admin bertugas mengelola data kamar, reservasi, pengguna, dan konten website melalui dashboard admin.

## Fitur Utama

### Fitur User
- Registrasi dan login pengguna
- Melihat informasi kamar dan layanan
- Melakukan reservasi kamar
- Pembayaran menggunakan sistem sandbox (simulasi)
- Melihat riwayat booking
- Check-in online
- Pengajuan pembatalan reservasi
- Mengelola profil pengguna
- Membaca artikel blog dan informasi website

### Fitur Admin
- Login admin
- Dashboard admin (statistik reservasi dan pembatalan)
- Manajemen data kamar dan layanan
- Manajemen data booking
- Manajemen pembatalan reservasi
- Manajemen user dan customer
- Manajemen konten blog
- Log aktivitas dan pesan

## Teknologi yang Digunakan

- Framework: Laravel
- Bahasa Pemrograman: PHP
- Database: MySQL
- Web Server & Environment: Laragon
- Frontend: HTML, CSS, JavaScript, dan framework pendukung

## Lingkup Proyek

Sistem ini difokuskan pada satu jenis layanan, yaitu akomodasi. Proses pembayaran dan pengembalian dana masih bersifat simulasi (sandbox). Aplikasi dirancang responsif dan dapat diakses melalui perangkat desktop maupun mobile.
